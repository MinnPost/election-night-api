Election Night API

A project of MinnPost

If you work at a small organization without access to AP or Reuters election data feeds, or are looking for more local results than these organizations provide, the Election Night API is here to help.

The purpose of this software is to make election results published by a state's Secretary of State or Board of Elections on election night available for use on websites or in other applications via an API.

This software will work best in states that publish election results live on election night in somewhat machine-friendly formats, like JSON, text files or maybe even Microsoft Excel.

Detailed setup instructions can be found in the README, but the general process for setting this up is this:

Install the code and dependencies

Election Night API is built in Python with architecture based on ScraperWiki. It can be run in three ways:

  • Locally, for development purposes
  • On a dedicated server, for election night
  • On ScraperWiki, for after the election when results will not be updated frequently

Write the scraper

Election Night API contains a basic template for scraping election night results files, but because each state and election is pretty different, custom scripts for parsing the results files must be written. You'll need to be able to write some basic Python to build a scraper for your state. There's some basic guidance in the README and in the example code.

Deploy the API

The API is intended to be deployed on a resourceful server, such as an Ubuntu server on EC2, on election night, to be able to handle high demand for the results. Detailed instructions can be found in the README.

Display results

Although the main purpose of this software is to provide API access to election results, the usual reason for doing this is to then display those results on a web page. For an example of one way to do that, see the demo.

Move to ScraperWiki

When election night is over and demand is lessened results are no longer being updated frequently, it is easy to move the API off of your server and on to ScraperWiki, which has a free tier for journalists.
